Tendência Central

Média

Média é uma medida que calcula o valor médio de um conjunto, em ciência de dados, a média é utilizada uma análise para obter uma medida de tendência central, ou seja, ela busca entender o valor fornecendo uma ideia de centralidade. Geralmente, é combinada com outras técnicas estatísticas, como a mediana e a moda, para uma compreensão mais completa dos dados.

Mediana

A mediana é calculada pelo valor que está no meio do conjunto quando eles estão ordenados de maneira crescente ou decrescente, já a média não leva em conta o ordenamento dos elementos do conjunto

Moda

A moda é uma medida estatística que busca os valores com a maior frequência no conjundo de dados, ela não é uma medida de tendência central (porém estudamos a moda juntamente com esses conteúdos) ela apenas faz uma contagem do elemento modal. Ela pode ser: Unimodal, Bimodal, Multimodal e Amodal

Percentil

Percentil refere-se a uma medida estatística que indica a posição relativa de um determinado valor em relação a um conjunto de valores. Em bancos de dados, o percentil é frequentemente usado para analisar a distribuição dos dados e identificar valores atípicos.

Quartil

Os quartis são medidas estatísticas que dividem um conjunto de dados ordenados em quatro partes iguais. Eles são amplamente utilizados para entender a distribuição e a dispersão dos dados. Os três quartis mais comuns são: Primeiro Quartil, Segundo Quartil e Terceiro Quartil.
Primeiro Quartil (Q1)
O primeiro quartil divide os dados em 25% (ou um quarto) inferior e 75% superior. Em termos de posição, o primeiro quartil corresponde à mediana dos dados inferiores.
Segundo Quartil (Q2)
O segundo quartil é exatamente igual à mediana dos dados. Também é conhecido como quartil médio ou quartil 50%. Divide os dados em 50% inferior e 50% superior.
Terceiro Quartil (Q3)
Conhecido como quartil superior ou quartil 75%, o terceiro quartil divide os dados em 75% inferior e 25% superior. O terceiro quartil é equivalente à mediana dos dados superiores.

Amplitude

A amplitude refere-se à diferença entre o maior e o menor valor em um conjunto de dados. É uma medida simples que fornece uma ideia da extensão total dos valores observados. Para calcular a amplitude, basta subtrair o menor valor do maior valor encontrado no conjunto de dados.

Desvio Padrão

O desvio padrão é uma medida estatística que indica a dispersão ou variabilidade dos dados em relação à média. Ele fornece uma medida de quão distantes os valores individuais estão da média do conjunto de dados.
