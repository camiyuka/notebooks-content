HTML

Html é uma linguagem de marcação de texto, utilizada para criação da estrutura básica de páginas web, permitindo que os navegadores interpretem e exibam as informações aos usuários. No primeiro semestre aprendemos o básico de HTML e CSS e criamos um currículo, com link para uma página "sobre mim".

CSS

CSS, ou Cascading Style Sheets (Folhas de Estilo em Cascata), é uma linguagem de estilo utilizada para definir a aparência e o layout de elementos HTML em páginas da web. Ele permite controlar cores, fontes, tamanhos, espaçamento e outros aspectos visuais. Java
