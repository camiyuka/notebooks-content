HTML

Html é uma linguagem de marcação de texto, utilizada para criação da estrutura básica de páginas web, permitindo que os navegadores interpretem e exibam as informações aos usuários. No primeiro semestre aprendemos o básico de HTML e CSS e criamos um currículo online, com link para uma página "Sobre mim".

CSS

CSS, ou Cascading Style Sheets (Folhas de Estilo em Cascata), é uma linguagem de estilo utilizada para definir a aparência e o layout de elementos HTML em páginas da web. Ele permite controlar cores, fontes, tamanhos, espaçamento e outros aspectos visuais. Como já dito anteriormente, fizemos o currículo, onde foi utilizado CSS, segue outro projeto utilizando HTML e CSS:
Projeto individual, onde eu refiz o meu primeiro site, tendo um pouco mais de noção de HTML e CSS.

JavaScript

Em JavaScript, aprendemos os fundamentos da linguagem, fizemos uso de estruturas de controle, como condicionais e loops, para tomar decisões e repetir ações. Aprendemos a manipular eventos e a interagir com elementos HTML por meio da manipulação do DOM. Como por exemplo: ao clicar em um botão, ao pressionar uma tecla, ao passar o mouse, entre outros. Nós fizemos vários projetos em JS, como a calculadora de impostos, um transformador de código e vários exercícios de operações matemáticas. O projeto abaixo é um gerador de cores aleatórias, que manipula as cores do HTML ao clicar no botão.

Java

Em Java, aprendemos a sintaxe básica da linguagem, como declarar variáveis, criar métodos e classes. Também aprendemos a usar estruturas de controle, como condicionais e loops, para tomar decisões e repetir ações. Entendemos como manipular objetos, criar hierarquias de classes e reutilizar código com encapsulamento, herança e polimorfismo. Além disso, aprendemos a lidar com exceções e a utilizar bibliotecas padrão.
O projeto abaixo, nós utilizamos o JFrame, da biblioteca Swing, permitindo-nos a criar uma janela e colocar uma imagem PNG, definindo no código que ela não pode passar da altura e largura da janela.
Também aprendemos C em automação industrial e Python em ciência de dados, que serão citados nos próximos conteúdos.
