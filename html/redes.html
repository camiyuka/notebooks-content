Tipos de redes e topologia

Tipos de Rede

PAN

Personal Area Network É uma rede de área pessoal que conecta dispositivos eletrônicos próximos a uma pessoa, geralmente em um alcance de até alguns metros. Exemplos de tecnologias de PAN incluem Bluetooth e infravermelho, que permitem a comunicação entre dispositivos como smartphones, tablets, fones de ouvido sem fio, teclados, etc.

CAN

Controller Area Network É uma rede de área de controle utilizada principalmente na indústria automotiva. O CAN é projetado para permitir a comunicação entre os diferentes dispositivos eletrônicos presentes em um veículo, como sensores, atuadores, unidades de controle do motor, painéis de instrumentos, etc. Ele permite uma troca rápida e confiável de dados, garantindo o funcionamento adequado do veículo.

LAN

Local Area Network (infinitas) É uma rede de área local que abrange uma área geograficamente limitada, como uma residência, escritório, escola ou campus. As LANs são usadas para conectar dispositivos próximos entre si, como computadores, impressoras, servidores e outros dispositivos de rede. Elas geralmente são construídas com tecnologias Ethernet ou Wi-Fi e oferecem altas velocidades de transmissão de dados dentro de sua área de cobertura.

MAN

Metropolitan Area Network (como TIM, VIVO, etc. Há inúmeras) É uma rede de área metropolitana que abrange uma área geográfica maior do que uma LAN, como uma cidade ou uma região metropolitana. As MANs são normalmente operadas por provedores de serviços de telecomunicações, como TIM, VIVO, entre outros, e conectam várias LANs e outros dispositivos em uma área geográfica específica. Elas fornecem velocidades de transmissão de dados mais altas do que as LANs, mas geralmente são mais limitadas em termos de alcance geográfico.

WAN

Wide Area Network (interconexões de MAN, apenas uma) É uma rede de área ampla que cobre uma grande área geográfica, como um país, continente ou até mesmo globalmente. As WANs são utilizadas para interconectar várias redes menores, como LANs e MANs, através de tecnologias de telecomunicação, como linhas telefônicas, cabos de fibra óptica ou conexões sem fio. A Internet é um exemplo de uma WAN global que permite a comunicação e o compartilhamento de informações em todo o mundo.

Comunicação

Firewall

É uma medida de segurança que controla e monitora o tráfego de rede, protegendo uma rede privada contra acesso não autorizado e ameaças externas. Ele desempenha um papel vital na prevenção de ataques e na manutenção da integridade e confidencialidade dos dados em uma rede.

MAC

Sigla para "Media Access Control", é responsável por controlar o acesso ao meio físico compartilhado e garantir uma comunicação eficiente entre os dispositivos de uma rede local, utilizando endereços MAC exclusivos para identificar cada dispositivo.

Unicast, Broadcast e Multicast

Unicast
Comunicação de um para um, a mensagem é endereçada especificamente para um único dispositivo na rede, identificado por seu endereço IP ou endereço MAC. A mensagem não é replicada ou compartilhada com outros dispositivos na rede.
Broadcast
Comunicação de um para todos, o remetente envia a mensagem para um endereço de broadcast especial, como o endereço de IP 255.255.255.255, e todos os dispositivos conectados à rede recebem a mensagem.É útil para enviar informações de forma geral ou para alcançar todos os dispositivos em uma rede local.
Multicast
Comunicação de um para muitos, o remetente envia a mensagem para um endereço de multicast específico, que representa um grupo de dispositivos interessados em receber a mensagem. A mensagem é replicada e enviada somente aos dispositivos que se inscreveram no grupo multicast.
