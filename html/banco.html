SQL

O SQL (Structured Query Language) é a linguagem de pesquisa declarativa padrão para banco de dados relacional, que pode ser divididas em: DDL: Linguagem de definição de dados; DML: Linguagem de manipulação de dados; DCL: Linguagem de Controle de Dados; TCL: Linguagem de controle de transação; DQL: linguagem de consulta de dados.
Em banco de dados, as linguagens que mais aprendemos foram: DML, DQL e DCL

Normalização de Banco de dados

1ª Forma Normal

Passe o mouse para saber mais

A 1FN estabelece que cada tabela em um banco de dados deve ter valores atômicos, cada valor em uma coluna deve ser indivisível.

2ª Forma Normal

Passe o mouse para saber mais

A 2FN estabelece que uma tabela deve eliminar dependências parciais:cada coluna em uma tabela deve depender completamente de sua chave primária.

3ª Forma Normal

Passe o mouse para saber mais

A 3FN estabelece que cada tabela deve eliminar dependências transitivas: todas as colunas não chave de uma tabela devem depender apenas da chave primária da tabela.
